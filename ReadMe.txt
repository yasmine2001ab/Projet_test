# Compilation du programme
Pour exécuter le programme, assurez-vous d'avoir les librairies requises pour la compilation dans Eclipse, Pour ça :
- les librairies se trouve dans le dossier libs dans le répertoire racine.
- Ouvrez Eclipse
- Importez le projet
- Cliquez droit sur le Répertoire racine "ABBAS_METTOUCHI_SAGHIR-L3_I-ProjetIntegration_TousSportsIndividuels"
- Build path/Configure Build path/Libraries/Classpath
- Vérifier si elles sont bien là.
- Sinon add external jar's qui se trouve dans le dossier libs
- Appliquer et fermer
